## Build & Test
### Prerequisites
* C++17 toolchain (GCC 9+/Clang 10+/MSVC 2019+)
* CMake 3.12+

### Clone
```
git clone https://github.com/GwangrokBaek/atm-controller atm-controller
```

### Build
```
cd atm-controller
cmake -S . -B build
cmake --build build -j
```

### Test
```
.\build\Debug\atm.exe
```
